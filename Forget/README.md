
# Forget Frontend


This is the front-end project made it with:

```
Ui: React
Css framework: Tailwind css
Components: Tremor, React-hot-toast
Api Management: Axios
Bundler: Vite
Context: Zustand

```

- [Tremor](https://www.tremor.so/)
- [React-hot-toast](https://react-hot-toast.com/)
