# Forget Project

This is the complete forget project, this is an E-Commerce, with authentication and role management for users

## Tech Stack

```
- React
- .Net
- SQL Server
- Postgres
- Vite
- Tailwind CSS
- TypeScript
- Axios
- C#
- Express
- Node
- Jwt
- Swagger
- Entity Framework
- Sequelize
```
